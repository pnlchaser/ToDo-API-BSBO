# ToDo API с матрицей Эйзенхауэра

## Описание
API для управления задачами с классификацией по матрице Эйзенхауэра (Q1-Q4)

## Технологии
- Python 3.11
- FastAPI
- Uvicorn

## Запуск проекта
1. Клонируйте репозиторий
2. Создайте виртуальное окружение: `python -m venv venv`
3. Активируйте окружение: `source venv/Scripts/activate`
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите сервер: `uvicorn main:app --reload`
